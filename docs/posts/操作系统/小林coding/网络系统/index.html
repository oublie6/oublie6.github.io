什么是零拷贝

减少「用户态与内核态的上下文切换」和「内存拷贝」的次数

两种实现方式

  1. mmap+write
  1. sendfile
  1. 真正的零拷贝技术:网卡支持 SG-DMA（The Scatter-Gather Direct Memory Access）技术（和普通的 DMA 有所不同）

什么是一致性哈希

提高均衡度：将虚拟节点映射到哈希环上，并将虚拟节点映射到实际节点，所以这里有「两层」映射关系
