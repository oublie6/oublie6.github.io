三色标记法

  1. 将所有对象标记为白色
  2. 将从root对象可达的对象标记为灰色然后放到队列中
  3. 从队尾取对象标记为黑色，将其引用的白色对象标记为灰色并放入队头，重复直到队列没有对象
  4. 清除白色对象
