本系列是作者在跟着W3school的JS教程学习时做的笔记，可能并不详尽，读者可以到W3school教程中查看完整内容。

addEventListener() 方法

添加一个在用户单击按钮时触发的事件侦听器：

document.getElementById("myBtn").addEventListener("click", displayDate);

addEventListener()方法将事件处理程序附加到元素而不覆盖现有的事件处理程序。

可以将多个相同类型的事件处理程序添加到一个元素，即两个“单击”事件。

可以将事件侦听器添加到任何 DOM 对象，而不仅仅是 HTML 元素。即可以添加到window对象。

addEventListener()方法可以更轻松地控制事件对冒泡的反应。

使用该addEventListener()方法时，JavaScript 与 HTML 标记分离，以提高可读性，并允许您添加事件侦听器，即使您不控制 HTML 标记。

可以使用该removeEventListener()方法轻松删除事件侦听器。

element.addEventListener(event, function, useCapture);

第一个参数是事件的类型（如 " click" 或 " mousedown" 或任何其他HTML DOM Event。）

第二个参数是我们要在事件发生时调用的函数。

第三个参数是一个布尔值，指定是使用事件冒泡(来自子元素)还是事件捕获（来自自身）。此参数是可选的。

请注意，您不要为事件使用“on”前缀；用“ click”代替“ onclick”。

将事件处理程序添加到元素

警报“Hello World！” 当用户点击一个元素时：

element.addEventListener("click", function(){ alert("Hello World!"); });

您还可以引用外部“命名”函数：

element.addEventListener("click", myFunction);

function myFunction() {
  alert ("Hello World!");
}

将多个事件处理程序添加到同一元素

addEventListener()方法允许您将许多事件添加到同一个元素，而不会覆盖现有事件：

element.addEventListener("click", myFunction);
element.addEventListener("click", mySecondFunction);

您可以将不同类型的事件添加到同一元素：

element.addEventListener("mouseover", myFunction);
element.addEventListener("click", mySecondFunction);
element.addEventListener("mouseout", myThirdFunction);

将事件处理程序添加到窗口对象

addEventListener()方法允许您在任何 HTML DOM 对象上添加事件侦听器，例如 HTML 元素、HTML 文档、窗口对象或其他支持事件的对象，如xmlHttpRequest对象。

添加一个在用户调整窗口大小时触发的事件监听器：

window.addEventListener("resize", function(){
  document.getElementById("demo").innerHTML = sometext;
});

传递参数

传递参数值时，使用“匿名函数”，使用参数调用指定函数：

element.addEventListener("click", function(){ myFunction(p1, p2); });

事件冒泡还是事件捕获？

HTML DOM 中有两种事件传播方式，冒泡和捕获。

事件传播是一种在事件发生时定义元素顺序的方法。如果你在 <div> 元素中有一个 <p> 元素，并且用户点击了 <p> 元素，那么应该首先处理哪个元素的“点击”事件？

在冒泡中，最内层元素的事件首先被处理，然后是最外层：首先处理 <p> 元素的点击事件，然后是 <div> 元素的点击事件。

在捕获最外层元素的事件时，首先处理内部：<div> 元素的单击事件将首先处理，然后是 <p> 元素的单击事件。

addEventListener() 方法，您可以使用“useCapture”参数指定传播类型：

addEventListener(event, function, useCapture);

默认值为false，将使用冒泡传播，当该值设置为true时，事件使用捕获传播。

document.getElementById("myP").addEventListener("click", myFunction, true);
document.getElementById("myDiv").addEventListener("click", myFunction, true);

removeEventListener() 方法

removeEventListener()方法删除已使用 addEventListener() 方法附加的事件处理程序：

element.removeEventListener("mousemove", myFunction);
