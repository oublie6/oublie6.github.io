HTMLCollection 对象

getElementsByTagName()方法返回一个HTMLCollection对象。

对象是 HTML 元素的HTMLCollection类似数组的列表（集合）。

以下代码选择文档中的所有元素<p>：

const myCollection = document.getElementsByTagName("p");

集合中的元素可以通过索引号访问。

要访问第二个 <p> 元素，您可以编写：myCollection[1]

注意：索引从 0 开始。

HTMLCollection 长度

length属性定义 中的元素数量HTMLCollection：

当您想要遍历集合中的元素时，length属性很有用：

const myCollection = document.getElementsByTagName("p");
for (let i = 0; i < myCollection.length; i++) {
  myCollection[i].style.color = "red";
}

HTMLCollection 不是数组！

HTMLCollection 可能看起来像一个数组，但它不是。

您可以遍历列表并使用数字引用元素（就像数组一样）。

但是，您不能在 HTMLCollection 上使用 valueOf()、pop()、push() 或 join() 等数组方法。
