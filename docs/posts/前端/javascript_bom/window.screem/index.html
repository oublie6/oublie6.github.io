window.screen 对象包含有关用户屏幕的信息。

window screen

可以在没有window前缀的情况下编写window.screen对象。

属性：

  • screen.width 以像素为单位返回访问者屏幕的宽度。
  • screen.height 返回访问者屏幕的高度（以像素为单位）。
  • screen.availWidth 返回访问者屏幕的宽度，以像素为单位，减去 Windows 任务栏等界面功能。
  • screen.availHeight 返回访问者屏幕的高度，以像素为单位，减去 Windows 任务栏等界面功能。
  • screen.colorDepth 返回用于显示一种颜色的位数。
  • screen.pixelDepth 返回屏幕的像素深度。

对于现代计算机，颜色深度和像素深度是相等的。
