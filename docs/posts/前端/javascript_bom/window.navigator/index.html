window.navigator对象包含有关访问者浏览器的信息。

window.navigator

可以在没有window前缀的情况下编写window.navigator对象。

一些例子：

  • navigator.cookieEnabled 如果启用了 cookie，则该cookieEnabled属性返回 true，否则返回 false：
  • navigator.appCodeName 返回浏览器的应用程序代码名称：此属性在最新的 Web 标准中已删除（已弃用）。大多数浏览器（IE11、Chrome、Firefox、Safari、Opera）将Mozilla作为 appCodeName 返回。
  • navigator.appName 返回浏览器的应用程序名称：此属性在最新的 Web 标准中已删除（已弃用）。大多数浏览器（IE11、Chrome、Firefox、Safari）将Netscape作为 appName 返回。
  • navigator.product 返回浏览器引擎的产品名称：此属性在最新的 Web 标准中已删除（已弃用）。大多数浏览器将Gecko作为产品返回。
  • navigator.appVersion 返回有关浏览器的版本信息：
  • navigator.userAgent 返回浏览器发送给服务器的用户代理头：
  • navigator.platform 返回浏览器平台（操作系统）：
  • navigator.language 返回浏览器的语言
  • navigator.onLine 如果浏览器在线，返回 true：
  • navigator.javaEnabled() 如果启用了Java ，返回 true ：

警告 ！！！

来自 navigator 对象的信息通常会产生误导，不应用于检测浏览器版本，因为：

  • 不同的浏览器可以使用相同的名称
  • 浏览器所有者可以更改导航器数据
  • 一些浏览器错误识别自己以绕过站点测试
  • 浏览器无法报告新的操作系统，发布晚于浏览器
