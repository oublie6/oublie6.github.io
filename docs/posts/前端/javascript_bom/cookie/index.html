Cookies 让您将用户信息存储在网页中。

什么是 Cookie？

Cookie 是存储在您计算机上的小型文本文件中的数据。

当 Web 服务器将网页发送到浏览器时，连接会关闭，服务器会忘记有关用户的所有信息。

发明 Cookie 是为了解决“如何记住用户信息”的问题：

  • 当用户访问网页时，他/她的姓名可以存储在 cookie 中。
  • 下次用户访问该页面时，cookie 会“记住”他/她的名字。

Cookie 保存在名称-值对中，例如：username = John Doe

当浏览器从服务器请求网页时，属于该页面的 cookie 会添加到请求中。通过这种方式，服务器获取必要的数据来“记住”有关用户的信息。

如果您的浏览器关闭了本地 cookie 支持，则以下示例均无效。

使用 JavaScript 创建 Cookie

JavaScript 可以使用document.cookie属性创建、读取和删除 cookie。

使用 JavaScript，可以像这样创建 cookie：

document.cookie = "username=John Doe";

您还可以添加到期日期（以 UTC 时间表示）。默认情况下，关闭浏览器时会删除 cookie：

document.cookie = "username=John Doe; expires=Thu, 18 Dec 2013 12:00:00 UTC";

使用路径参数，您可以告诉浏览器 cookie 属于哪个路径。默认情况下，cookie 属于当前页面。

document.cookie = "username=John Doe; expires=Thu, 18 Dec 2013 12:00:00 UTC; path=/";

使用 JavaScript 读取 Cookie

使用 JavaScript，cookie 可以这样读取：let x = document.cookie;

document.cookie将在一个字符串中返回所有 cookie，就像：cookie1=value; cookie2=值；cookie3=值；

使用 JavaScript 更改 Cookie

使用 JavaScript，您可以像创建 cookie 一样更改 cookie：

document.cookie = "username=John Smith; expires=Thu, 18 Dec 2013 12:00:00 UTC; path=/";

旧的 cookie 被覆盖。

使用 JavaScript 删除 Cookie

删除 cookie 非常简单。

删除 cookie 时不必指定 cookie 值。

只需将 expires 参数设置为过去的日期：

document.cookie = "username=; expires=Thu, 01 Jan 1970 00:00:00 UTC; path=/;";

您应该定义 cookie 路径以确保删除正确的 cookie。

如果您不指定路径，某些浏览器将不允许您删除 cookie。

Cookie 字符串

document.cookie属性看起来像一个普通的文本字符串。但事实并非如此。

即使你将整个cookie字符串写入document.cookie，当你再次读出它时，你也只能看到它的名称-值对。

如果您设置了新的 cookie，旧的 cookie 不会被覆盖。新的 cookie 被添加到 document.cookie 中，所以如果你再次阅读 document.cookie，你会得到类似的东西：

如果要查找某个指定 cookie 的值，则必须编写一个 JavaScript 函数来搜索 cookie 字符串中的 cookie 值。

JavaScript Cookie 示例

在接下来的示例中，我们将创建一个存储访问者姓名的 cookie。

访问者第一次到达该网页时，会要求他/她填写他/她的姓名。然后将该名称存储在 cookie 中。

下次访问者到达同一页面时，他/她将收到一条欢迎消息。

对于示例，我们将创建 3 个 JavaScript 函数：

  1. 设置 cookie 值的函数
  2. 获取 cookie 值的函数
  3. 检查 cookie 值的函数

设置 Cookie 的函数

首先，我们创建一个function将访问者姓名存储在 cookie 变量中：

function setCookie(cname, cvalue, exdays) {
  const d = new Date();
  d.setTime(d.getTime() + (exdays*24*60*60*1000));
  let expires = "expires="+ d.toUTCString();
  document.cookie = cname + "=" + cvalue + ";" + expires + ";path=/";
}

示例解释：

上述函数的参数是 cookie 的名称 (cname)、cookie 的值 (cvalue) 以及 cookie 应过期的天数 (exdays)。

该函数通过将 cookie 名称、cookie 值和过期字符串相加来设置 cookie。

获取 Cookie 的函数

然后，我们创建一个function返回指定 cookie 的值：

function getCookie(cname) {
  let name = cname + "=";
  let decodedCookie = decodeURIComponent(document.cookie);
  let ca = decodedCookie.split(';');
  for(let i = 0; i <ca.length; i++) {
    let c = ca[i];
    while (c.charAt(0) == ' ') {
      c = c.substring(1);
    }
    if (c.indexOf(name) == 0) {
      return c.substring(name.length, c.length);
    }
  }
  return "";
}

函数说明：

将 cookiename 作为参数 (cname)。

使用要搜索的文本创建一个变量 (name) (cname + “=")。

解码 cookie 字符串，以处理带有特殊字符的 cookie，例如 ‘$’

将分号上的 document.cookie 拆分为一个名为 ca (ca = decodedCookie.split(’;’)) 的数组。

循环遍历 ca 数组 (i = 0; i < ca.length; i++)，并读出每个值 c = ca[i])。

如果找到 cookie (c.indexOf(name) == 0)，则返回 cookie 的值 (c.substring(name.length, c.length)。

如果未找到 cookie，则返回“”。

检查 Cookie 的函数

最后，我们创建检查是否设置了 cookie 的函数。

如果设置了 cookie，它将显示一个问候语。

如果没有设置cookie，会显示一个提示框，询问用户名，并将用户名cookie保存365天，调用setCookie函数：

function checkCookie() {
  let username = getCookie("username");
  if (username != "") {
   alert("Welcome again " + username);
  } else {
    username = prompt("Please enter your name:", "");
    if (username != "" && username != null) {
      setCookie("username", username, 365);
    }
  }
}
