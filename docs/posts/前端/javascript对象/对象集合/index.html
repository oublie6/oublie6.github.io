如何创建集合

您可以通过以下方式创建 JavaScript 集：

  • 将数组传递给new Set()
  • 创建一个新的 Set 并用于add()添加值
  • 创建一个新的 Set 并用于add()添加变量

new Set()方法

// Create a Set
const letters = new Set(["a","b","c"]);

add() 方法

letters.add("d");
letters.add("e");

forEach() 方法

该forEach()方法为每个 Set 元素调用一个函数：

// Create a Set
const letters = new Set(["a","b","c"]);

// List all entries
let text = "";
letters.forEach (function(value) {
  text += value;
})

values() 方法

values()方法返回一个包含 Set 中所有值的 Iterator 对象：

letters.values()   // Returns [object Set Iterator]

现在您可以使用 Iterator 对象来访问元素：

// Create an Iterator
const myIterator = letters.values();

// List all Values
let text = "";
for (const entry of myIterator) {
  text += entry;
}

keys()方法

Set 没有键。

keys()返回与values()相同 。

这使得集合与Map兼容。

letters.keys()   // Returns [object Set Iterator]

entry() 方法

Set 没有键。

entries()返回 [value,value] 对而不是 [key,value] 对。

这使得 Sets 与 Maps 兼容：

// Create an Iterator
const myIterator = letters.entries();

// List all Entries
let text = "";
for (const entry of myIterator) {
  text += entry;
}

集合是对象

对于 Set，typeof返回对象：typeof letters; // Returns object

对于 Set，instanceof Set返回 true：letters instanceof Set; // Returns true
