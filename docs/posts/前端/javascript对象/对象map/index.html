如何创建Map

您可以通过以下方式创建 JavaScript 地图：

  • 将数组传递给new Map()
  • 创建Map并使用Map.set()

new Map()

您可以通过将 Array 传递给new Map()构造函数来创建 Map：

// Create a Map
const fruits = new Map([
  ["apples", 500],
  ["bananas", 300],
  ["oranges", 200]
]);

set()

您可以使用set()方法将元素添加到 Map ：

// Create a Map
const fruits = new Map();

// Set Map Values
fruits.set("apples", 500);
fruits.set("bananas", 300);
fruits.set("oranges", 200);

Map.forEach()

该forEach()方法为 Map 中的每个键/值对调用回调：

// List all entries
let text = "";
fruits.forEach (function(value, key) {
  text += key + ' = ' + value;
})

entries()

entries()方法在 Map 中返回一个带有 [key,values] 的迭代器对象：

// List all entries
let text = "";
for (const x of fruits.entries()) {
  text += x;
}

keys()

keys()方法返回一个迭代器对象，其中包含 Map 中的键：

// List all keys
let text = "";
for (const x of fruits.keys()) {
  text += x;
}

values()

values()方法返回一个迭代器对象，其中包含 Map 中的值：

// List all values
let text = "";
for (const x of fruits.values()) {
  text += x;
}

对象作为key

能够将对象用作键是 Map 的一项重要功能。

// Create Objects
const apples = {name: 'Apples'};
const bananas = {name: 'Bananas'};
const oranges = {name: 'Oranges'};

// Create a Map
const fruits = new Map();

// Add new Elements to the Map
fruits.set(apples, 500);
fruits.set(bananas, 300);
fruits.set(oranges, 200);

请记住：键是一个对象（apples），而不是一个字符串（“apples”）：

fruits.get("apples");  // Returns undefined
