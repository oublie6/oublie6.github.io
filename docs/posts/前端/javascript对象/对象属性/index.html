JavaScript 属性

属性是与 JavaScript 对象关联的值。

JavaScript 对象是无序属性的集合。

属性通常可以更改、添加和删除，但有些是只读的。

访问 JavaScript 属性

objectName.property      // person.age

objectName["property"]   // person["age"]

objectName[expression]   // x = "age"; person[x]

表达式必须计算为属性名称。

JavaScript for…in 循环

for…in语句循环遍历对象的属性。

const person = {
  fname:" John",
  lname:" Doe",
  age: 25
};

for (let x in person) {
  txt += person[x];
}

添加新属性

可以通过简单地给现有对象一个值来向现有对象添加新属性。

假设 person 对象已经存在——然后你可以给它新的属性：person.nationality = "English";

删除属性

关键字delete从对象中删除属性：

const person = {
  firstName: "John",
  lastName: "Doe",
  age: 50,
  eyeColor: "blue"
};

delete person.age;

or delete person[“age”];

关键字delete会删除属性的值和属性本身。

删除后，该属性在重新添加之前无法使用。

delete运算符旨在用于对象属性。它对变量或函数没有影响。

delete运算符不应用于预定义的 JavaScript 对象属性。它可能会使您的应用程序崩溃。

嵌套对象

一个对象中的值可以是另一个对象：

myObj = {
  name:"John",
  age:30,
  cars: {
    car1:"Ford",
    car2:"BMW",
    car3:"Fiat"
  }
}

您可以使用点表示法或方括号表示法访问嵌套对象：myObj.cars.car2;,myObj["cars"]["car2"];或者myObj.cars["car2"];

嵌套数组和对象

对象中的值可以是数组，数组中的值可以是对象：

const myObj = {
  name: "John",
  age: 30,
  cars: [
    {name:"Ford", models:["Fiesta", "Focus", "Mustang"]},
    {name:"BMW", models:["320", "X3", "X5"]},
    {name:"Fiat", models:["500", "Panda"]}
  ]
}

要访问数组中的数组，请对每个数组使用 for-in 循环：

for (let i in myObj.cars) {
  x += "<h1>" + myObj.cars[i].name + "</h1>";
  for (let j in myObj.cars[i].models) {
    x += myObj.cars[i].models[j];
  }
}

属性的属性

所有属性都有一个name。此外，它们也具有value。

value是属性的属性之一。

其他属性有：enumerable可枚举、configurable可配置和writable可写。

这些属性定义了如何访问属性（它可读吗？它是否可写？）

在 JavaScript 中，可以读取所有属性，但只能更改 value 属性（并且仅当属性可写时）。

（ECMAScript 5 具有获取和设置所有属性的属性的方法）

原型属性

JavaScript 对象继承其原型的属性。

delete关键字不会删除继承的属性，但是如果删除原型属性，它将影响从原型继承的所有对象。
