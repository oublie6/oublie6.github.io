JavaScript 迭代器

迭代器协议定义了如何从一个对象产生一系列值。

对象在实现next()方法时成为迭代器。

该next()方法必须返回一个具有两个属性的对象：

  • value (the next value):The value returned by the iterator (Can be omitted if done is true)
  • done (true or false):true if the iterator has completed false if the iterator has produced a new value

自定义可迭代

这个可迭代的返回永无止境：10,20,30,40,…. 每次 next()都被调用：

// Home Made Iterable
function myNumbers() {
  let n = 0;
  return {
    next: function() {
      n += 10;
      return {value:n, done:false};
    }
  };
}

// Create Iterable
const n = myNumbers();
n.next(); // Returns 10
n.next(); // Returns 20
n.next(); // Returns 30

上面的对象是可迭代的：但是它不支持 JavaScript for..of语句。

JavaScript 可迭代对象是具有Symbol.iterator的对象。

Symbol.iterator是一个返回next()函数的函数。

可以使用代码迭代可迭代对象：for (const x of iterable) { }

// Create an Object
myNumbers = {};

// Make it Iterable
myNumbers[Symbol.iterator] = function() {
  let n = 0;
  done = false;
  return {
    next() {
      n += 10;
      if (n == 100) {done = true}
      return {value:n, done:done};
    }
  };
}

现在你可以使用for..of

for (const num of myNumbers) {
  // Any Code Here
}

Symbol.iterator 方法由for..of自动调用。

但我们也可以“手动”完成：

let iterator = myNumbers[Symbol.iterator]();

while (true) {
  const result = iterator.next();
  if (result.done) break;
  // Any Code Here
}
