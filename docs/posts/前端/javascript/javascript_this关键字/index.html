本系列是作者在跟着W3school的JS教程学习时做的笔记，可能并不详尽，读者可以到W3school教程中查看完整内容。

JavaScript 中，this关键字指的是一个对象。哪个对象取决于this调用（使用或调用）的方式。

this关键字根据其使用方式指代不同的对象：

  • 在对象方法中，this指的是对象。
  • 单独使用，this指的是全局对象(window)。
  • 在函数中，this指的是全局对象(window)。
  • 在严格模式下的函数中，this是undefined.
  • 在一个事件中，this指的是接收到该事件的Html元素。
  • 像call()、apply()和bind()这样的方法可以引用任何this对象。区别

this不是变量。它是一个关键字。您不能更改 的值this。

显式函数绑定

call()和apply()方法是预定义的 JavaScript 方法。

它们都可以用来调用以另一个对象为参数的对象方法。

const person1 = {
  fullName: function() {
    return this.firstName + " " + this.lastName;
  }
}

const person2 = {
  firstName:"John",
  lastName: "Doe",
}

// Return "John Doe":
person1.fullName.call(person2);

函数借用

使用该bind()方法，一个对象可以从另一个对象借用一个方法。

const person = {
  firstName:"John",
  lastName: "Doe",
  fullName: function () {
    return this.firstName + " " + this.lastName;
  }
}

const member = {
  firstName:"Hege",
  lastName: "Nilsen",
}

let fullName = person.fullName.bind(member);
