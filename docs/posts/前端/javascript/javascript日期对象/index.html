JavaScript 日期输出

默认情况下，JavaScript 将使用浏览器的时区并将日期显示为全文字符串：2022 年 10 月 5 日星期三 18:32:06 GMT+0800 (中国标准时间)

创建日期对象

日期对象是使用 new Date()构造函数创建的。有4 种方法可以创建新的日期对象：

  • new Date()
  • new Date(year, month, day, hours, minutes, seconds, milliseconds)
  • new Date(milliseconds)
  • new Date(date string)

new Date()

new Date()使用当前日期和时间创建一个新的日期对象：

上个世纪

一位和两位数的年份将被解释为 19xx：const d = new Date(99, 11, 24);

new Date(dateString)

new Date(dateString)从日期字符串创建一个新的日期对象：const d = new Date("October 13, 2014 11:13:00");
