如何创建字典

  • 将数组传递给new Map()
  • 创建Map并使用Map.set()

new Map()方法

// Create a Map
const fruits = new Map([
  ["apples", 500],
  ["bananas", 300],
  ["oranges", 200]
]);

set() 方法

// Create a Map
const fruits = new Map();

// Set Map Values
fruits.set("apples", 500);
fruits.set("bananas", 300);
fruits.set("oranges", 200);

set()方法还可用于更改现有的 Map 值：

get() 方法

get()方法获取 Map 中键的值：fruits.get("apples");

size属性

size属性返回 Map 中元素的数量：fruits.size;

delete() 方法

delete()方法删除一个 Map 元素：fruits.delete("apples");

has() 方法

fruits.has("apples");

JavaScript 对象与Map区别

见官网

forEach()

// List all entries
let text = "";
fruits.forEach (function(value, key) {
  text += key + ' = ' + value;
})

entry() 方法

entries()方法在 Map 中返回一个带有 [key, values] 的迭代器对象：

// List all entries
let text = "";
for (const x of fruits.entries()) {
  text += x;
}
