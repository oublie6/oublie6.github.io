JavaScript 字符串作为对象

通常，JavaScript 字符串是原始值，由固定值创建：let x = "John";

但是字符串也可以用关键字定义为对象new：let y = new String("John");

不要创建字符串对象。

关键字new使代码复杂化并减慢执行速度。

字符串对象会产生意想不到的结果：

使用==运算符时，x 和 y相等：

let x = "John";
let y = new String("John");

使用===运算符时，x 和 y不相等：

let x = "John";
let y = new String("John");

==会先进行类型转换，将后面的转换成前面的

===不会进行类型转换，类型不一样就直接为false
