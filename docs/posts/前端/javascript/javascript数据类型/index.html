JavaScript-JavaScript数据类型

本系列是作者在跟着W3school的JS教程学习时做的笔记，可能并不详尽，读者可以到W3school教程中查看完整内容。

JavaScript 变量可以保存不同的数据类型：数字、字符串、对象等等：

let length = 16;                               // Number
let lastName = "Johnson";                      // String
let x = {firstName:"John", lastName:"Doe"};    // Object

添加数字和字符串时，JavaScript 会将数字视为字符串。

JavaScript 从左到右计算表达式。

JavaScript 具有动态类型。这意味着可以使用相同的变量来保存不同的数据类型：

let x;           // Now x is undefined
x = 5;           // Now x is a Number
x = "John";      // Now x is a String

JavaScript 字符串

字符串是用引号写的。您可以使用单引号或双引号： 您可以在字符串中使用引号，只要它们与字符串周围的引号不匹配：

let answer1 = "It's alright";             // Single quote inside double quotes
let answer2 = "He is called 'Johnny'";    // Single quotes inside double quotes
let answer3 = 'He is called "Johnny"';    // Double quotes inside single quotes

JavaScript 数字

JavaScript 只有一种数字。 数字可以写成带小数，也可以不带小数： 超大或超小数可以用科学（指数）记数法书写：

let y = 123e5;      // 12300000
let z = 123e-5;     // 0.00123

JavaScript 布尔值

布尔值只能有两个值：true或false。

JavaScript 数组

JavaScript 数组是用方括号编写的。 数组项用逗号分隔。 数组索引是从零开始的，这意味着第一项是0，第二项是1，依此类推。

JavaScript 对象

JavaScript 对象是用花括号编写的{}。

对象属性写成名称：值对，用逗号分隔。

typeof 运算符

可以使用 JavaScripttypeof运算符来查找 JavaScript 变量的类型。

undefined

在 JavaScript 中，没有值的变量有值undefined。类型也是undefined。

空值

空值与undefined无关。空字符串同时具有合法值和类型。
