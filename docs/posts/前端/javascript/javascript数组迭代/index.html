JavaScript 数组 forEach()

该forEach()方法为每个数组元素调用一次函数（回调函数）。

请注意，该函数需要 3 个参数：

  • The item value
  • The item index
  • The array itself

可以仅使用前面几个

JavaScript 数组map（）

map()方法通过对每个数组元素执行一个函数来创建一个新数组。

map()方法不对没有值的数组元素执行该函数。

map()方法不会更改原始数组。

该函数同样需要 3 个参数

JavaScript 数组filter()

filter()方法使用通过测试的数组元素创建一个新数组。

该函数同样需要 3 个参数

JavaScript 数组 reduce()

reduce()方法在每个数组元素上运行一个函数以生成（将其减少为）单个值。

reduce()方法在数组中从左到右工作。另请参阅reduceRight()。

reduce()方法不会减少原始数组。

const numbers = [45, 4, 9, 16, 25];
let sum = numbers.reduce(myFunction);

function myFunction(total, value, index, array) {
  return total + value;
}

请注意，该函数需要 4 个参数：

  • The total (the initial value / previously returned value)
  • The item value
  • The item index
  • The array itself

reduce()方法可以接受一个初始值

const numbers = [45, 4, 9, 16, 25];
let sum = numbers.reduce(myFunction, 100);

function myFunction(total, value) {
  return total + value;
}

JavaScript 数组 every()

every()方法检查是否所有数组值都通过了测试。

const numbers = [45, 4, 9, 16, 25];
let allOver18 = numbers.every(myFunction);

function myFunction(value, index, array) {
  return value > 18;
}

JavaScript 数组 some()

some()方法检查某些数组值是否通过了测试。

const numbers = [45, 4, 9, 16, 25];
let someOver18 = numbers.some(myFunction);

function myFunction(value, index, array) {
  return value > 18;
}

JavaScript 数组 indexOf()

indexOf()方法在数组中搜索元素值并返回其位置。

array.indexOf(item, start)

JavaScript 数组 lastIndexOf()

Array.lastIndexOf()与 相同Array.indexOf()，但返回指定元素最后一次出现的位置。

array.lastIndexOf(item, start)

JavaScript 数组find()

find()方法返回通过测试函数的第一个数组元素的值。

const numbers = [4, 9, 16, 25, 29];
let first = numbers.find(myFunction);

function myFunction(value, index, array) {
  return value > 18;
}

JavaScript 数组 findIndex()

该findIndex()方法返回通过测试函数的第一个数组元素的索引。

const numbers = [4, 9, 16, 25, 29];
let first = numbers.findIndex(myFunction);

function myFunction(value, index, array) {
  return value > 18;
}

JavaScript Array.from()

Array.from()方法从具有长度属性的任何对象或任何可迭代对象返回一个 Array 对象。

Array.from("ABCDEFG");

JavaScript 数组keys()

Array.keys()方法返回一个带有数组键的 Array Iterator 对象。

JavaScript 数组 includes()

ECMAScript 2016 引入Array.includes()数组。这允许我们检查一个元素是否存在于数组中（包括 NaN，与 indexOf 不同）。

array.includes(search-item)

Array.includes() 允许检查 NaN 值。与 Array.indexOf() 不同。
