有 3 种方法可以提取字符串的一部分，索引都是左闭右开：

  • slice(start, end)//支持小于0的索引，从后往前数
  • substring(start, end) //小于0的索引被当作0
  • substr(start, length)//支持小于0的索引，从后往前数

替换字符串内容

replace()方法将指定值替换为字符串中的另一个值：

let text = "Please visit Microsoft!";
let newText = text.replace("Microsoft", "W3Schools");

replace()方法不会更改调用它的字符串。

replace()方法返回一个新字符串。

replace()方法仅替换第一个匹配项

如果要替换所有匹配项，请使用设置了 /g 标志的正则表达式。请参阅下面的示例。

要替换不区分大小写，请使用带有标志的正则表达式（不区分大小写）：/i

let text = "Please visit Microsoft!";
let newText = text.replace(/MICROSOFT/gi, "W3Schools");

正则表达式不带引号。

JavaScript 字符串 padStart()

ECMAScript 2017 添加了两个 String 方法：padStart() 并padEnd()支持在字符串的开头和结尾进行填充。

padStart()方法用另一个字符串填充一个字符串：

let text = "5";
let padded = text.padStart(4,"x");

该padStart()方法是一个字符串方法。要填充数字，请先将数字转换为字符串。

将字符串转换为数组

如果要将字符串用作数组，可以将其转换为数组。

JavaScript 字符串split（）

可以使用以下方法将字符串转换为数组split()：

text.split(",")    // Split on commas
text.split(" ")    // Split on spaces
text.split("|")    // Split on pipe

如果省略分隔符，则返回的数组将包含索引 [0] 中的整个字符串。 如果分隔符为 “"，则返回的数组将是单个字符的数组：text.split("")
