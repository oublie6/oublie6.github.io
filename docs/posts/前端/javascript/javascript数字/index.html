JavaScript 数字始终是 64 位浮点数

与许多其他编程语言不同，JavaScript 没有定义不同类型的数字，如整数、短整数、长整数、浮点数等。

遵循国际 IEEE 754 标准，JavaScript 数字始终存储为双精度浮点数。

这种格式以 64 位存储数字，其中数字（小数）存储在 0 到 51 位，指数存储在 52 到 62 位，符号存储在 63 位：

整数精度

整数（没有句点或指数符号的数字）精确到 15 位。最大小数位数为 17。

如果数字常量前面有 0x，JavaScript 会将其解释为十六进制。
